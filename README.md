## Introduction

**Default Detach** lets each user choose which application windows automatically open in a detached browser window using Foundry V14's built-in detach feature. System agnostic.

## Features

- **Per-User Configuration**: Each player picks their own auto-detach preferences
- **Runtime Discovery**: The config dialog lists every framed application opened during the session
- **Zero Setup**: Works out of the box with any system — no system-specific code

## Installation

Install directly through Foundry's module manager or manually using this manifest URL:
`https://github.com/Sayshal/default-detach/releases/latest/download/module.json`

## Usage

1. Open the applications you want to auto-detach at least once (actor sheets, item sheets, journals, etc.)
2. Go to **Module Settings > Default Detach > Configure**
3. Check the applications you want to always open in a detached window
4. Save — those application types will auto-detach on every future open
